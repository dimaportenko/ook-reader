# Milestone 1 — Foundation

[← Roadmap](../../roadmap.md)

**Goal:** be able to write Rust, build a Dioxus desktop app, and run/debug it — all from
NeoVim. Establish the single-codebase base that mobile/web extend later.

**Status:** ⬜ planned (current focus)

## Features

| # | Feature | Outcome | Status |
|---|---|---|---|
| 1 | [Dev Environment](01-dev-environment/README.md) | Rust/Dioxus NeoVim toolchain: LSP, build, run, debug | ⬜ |
| 2 | [Project Scaffold](02-project-scaffold/README.md) | `dx new` Dioxus desktop app that builds & runs | ⬜ |
| 3 | [Rust + Dioxus Fundamentals](03-rust-dioxus-fundamentals/README.md) | Comfortable with Rust + Dioxus basics | ⬜ |

## Exit criteria

- [ ] rust-analyzer attaches to `.rs` files in NeoVim
- [ ] `dx serve --platform desktop` runs a window
- [ ] App builds, runs, and stops at a breakpoint (`:RustLsp debuggables`) **from NeoVim**
- [ ] Comfortable enough with Rust/Dioxus to start the reader UI
